# DeepStorm: Deep Learning Framework

## Summary:
Deep Learning Framework from scratch in C++ using only the tensor class from libtorch.

## Layers & DL classes in framework:
- Conv2d
- MaxPool2d
- BatchNorm2d
- Flatten
- Dropout
- Linear
- ReLU
- Softmax
- SgdWithMomentum
- Adam
- CrossEntropyLoss
- Xavier
- He
